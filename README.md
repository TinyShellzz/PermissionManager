## 1.usage command
add permission  
```
/perm remove <playerName> <permissionNode>
/perm remove name minecraft.command.list
```
```
/perm add <playerName> <permissionNode>  
/perm add name minecraft.command.list
```

## 2.usage config file, edit groups.yml and uses.yml
reload config
```
/perm reload
```
